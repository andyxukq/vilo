We currently don’t offer direct exchanges—including size exchanges.

If you need a different size

  • Return your original order (within the trial window)
  • Place a new order for the correct size

    • We do this to keep fulfillment fast, inventory clean, and timelines predictable.

Want to avoid a second order?

Use the Sizing Guide (and sizer set) and wear your sizer for 24 hours, including sleep. If you’re between sizes, contact us before ordering—we’ll help you choose.
