Fit is everything.

The right size disappears on you—in the best way: secure, comfortable, and stable while you sleep.

The best method: use the Vilo Sizer Set

  1. Choose the finger you plan to wear Vilo on
  2. Wear a sizer for 24 hours, including sleep
  3. Pick the size that feels secure without discomfort

What “correct fit” feels like

  • It sits snugly without pinching
  • It doesn’t spin freely
  • It slides off with a bit of intent (not effort, not panic)

If you’re between sizes

  • Want a more secure fit (especially for sleep)? Size down
  • Fingers swell often / prefer easy removal? Size up

When to measure

Measure in the evening. Fingers can swell slightly throughout the day.
